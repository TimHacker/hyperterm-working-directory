# usage

- Open `~/.hyperterm.js`
- Add `hyperterm-working-directory` to the list of `plugins`
- Set `config.workingDirectory` to something like `~/dev`

That's it. If `config.workingDirectory` is not set, a default value of `$HOME` will be used.
